# Roadmap to v1.0.0

This roadmap outlines the planned progression from v0.10.0 to the public beta v1.0.0 release.

---

## v0.11.0 — Core POS Entities (Phase 1)
- Customer CRUD stabilization  
- Item CRUD + categories  
- Tax rate CRUD improvements  
- Sale + Payment save pipeline  
- POS Terminal session foundations  
- Consistency tests across modules  

---

## v0.12.0 — Inventory & Stock (Phase 2)
- Stock movements  
- Stock levels synchronization  
- Vendor and purchasing module foundation  
- Stock adjustments and admin inventory tools  
- Low-stock notifications  

---

## v0.13.0 — Accounting Layer
- Double-entry ledger skeleton  
- Sales journal  
- Cash drawer management and reconciliation  
- Financial data export routines  

---

## v0.14.0 — Developer Tools & Diagnostics
- SystemHub diagnostics integration  
- Developer onboarding script  
- Error clarity + user-friendly API messages  
- Health dashboards  
- Performance metrics  

---

## v0.15.0 — Security Hardening
- JWT rotation  
- Audit logging  
- Rate limiting  
- Suspicious activity monitors  
- Session-locking for terminals  

---

## v1.0.0 — Public Beta (Feature Complete)
- Full POS capability  
- Multi-org user workflows  
- Terminal operations  
- Inventory + Accounting stable  
- Full documentation freeze  
- macOS + Windows onboarding release  
- GitHub project public-ready  

---
